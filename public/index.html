Docker Node Webapp

A minimal Node.js web app served with Express and packaged with Docker.

What this demo shows

  • Serving static files with Express
  • Simple health API at /api/health
  • How to build and run the app with Docker

Try it

  1. Build the image: docker build -t docker-node-webapp .
  2. Run it locally: docker run -p 3000:3000 docker-node-webapp
  3. Open http://localhost:3000 in your browser

API check: /api/health

Created for teaching — simple, focused, and easy to extend.
